## Logging Benchmark
This is the repo for Automatic Logging Benchmark (AL-Bench).

This repo has two key parts, one is `./Static_Evaluation` and the other is `./Dynamic_Evaluation`.
In `./Static_Evaluation`, we present the evaluation scripts for each logging tool, and the evaluation reults we put into Google One Drive. 

In `./Dynamic_Evaluation`, we present all the scripts to execute the dynamic evaluation, the details are under `./Dynamic_Evaluation/README.md`, we provide a demo data for everyone to run this process. The whole process we used dynamic evaluation is presented in Google One Drive.

ALL the data could be found in https://drive.google.com/drive/u/1/folders/1eoK7SaYTuwqcAe9T3ddjeU5oGLRDX2Ps

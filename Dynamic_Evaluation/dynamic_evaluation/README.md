In `./data/demo/test/demo.json`. First replace the `function_position` with you actually path in you computer, and in `dynamic_evaluation.js` you need to replace the // path in docker // with your actual path in docker.

In your docker enviroment, run the scirpt in `statr-test.sh`, you could successfully run demo dynamic evaluation.
